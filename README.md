This repo makes aseprite easier to install!

**Linux**
  
  To install aseprite, run the following code:
  
  ```
  git clone --branch debian https://www.github.com/SmallCoder13/aseprite 
  
  cd aseprite && unzip aseprite-for-debian.zip
  
  cd aseprite_files
  
  ./aseprite_install
  ```
